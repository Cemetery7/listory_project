# BRAND_GUIDELINES.md

# Brand Guidelines

Документ описывает фирменный стиль проекта «Листория».

Все логотипы, цвета и брендовые элементы являются частью визуальной идентичности проекта.

Изменение фирменного стиля допускается только после утверждения владельцем проекта.

---

# 1. Brand Identity

Название проекта

```text
Листория
```

Слоган

```text
Истории рядом
```

Основной символ

```text
Открытая книга
```

Запрещается использовать:

- перо;
- чернильницу;
- символы, визуально напоминающие Ficbook;
- другие варианты логотипа.

---

# 2. Official Assets

Все официальные логотипы находятся в

```text
assets/
└── brand/
```

Структура

```text
brand/

logo-primary.png

logo-horizontal.png

logo-icon.png

logo-primary.svg

logo-horizontal.svg

logo-icon.svg
```

SVG является основным форматом.

PNG используется как запасной вариант.

# 10. Clear Space

Вокруг логотипа всегда оставлять свободную область.

Минимум

```text
0.5 × высота логотипа
```

Никакие элементы интерфейса не должны заходить в эту область.

---

# 11. Background

Темная тема

Использовать цветной логотип.

---

Светлая тема

Использовать стандартную цветную версию.

При необходимости допускается монохромная версия.

---

# 12. Scaling

Разрешается

- уменьшать;
- увеличивать;
- экспортировать SVG.

Запрещается

- изменять пропорции;
- растягивать;
- сжимать;
- поворачивать;
- наклонять.

---

# 13. Color Rules

Запрещается

- менять цвета книги;
- менять цвет свечения;
- менять фирменный фиолетовый;
- использовать случайные градиенты.

---

# 14. Prohibited

Запрещается

- менять форму книги;
- добавлять новые элементы;
- использовать перо;
- использовать другие иконки книг;
- применять тени, не предусмотренные дизайн-системой;
- использовать неофициальные версии логотипа.

---

# 15. SVG

Основной формат

```text
SVG
```

PNG используется только:

- в презентациях;
- если SVG не поддерживается;
- для предварительных изображений.

---

# 16. Codex Rules

Codex обязан использовать только официальные логотипы.

Запрещается

- генерировать новые логотипы;
- изменять существующий логотип;
- менять размеры исходных файлов;
- заменять логотип другими символами.

Разрешается только масштабирование.

---

# 17. Definition of Done

Брендинг считается реализованным, если:

- используется только официальный логотип;
- соблюдаются размеры;
- используются только официальные цвета;
- не нарушены правила масштабирования;
- используются SVG при возможности.
