# Brand Assets

Данная папка содержит официальные логотипы проекта «Листория».

Все изображения являются частью фирменного стиля проекта.

Использовать только эти файлы.

---

# Структура

```text
assets/

└── brand/

    logo-primary.svg
    logo-primary.png

    logo-horizontal.svg
    logo-horizontal.png

    logo-icon.svg
    logo-icon.png
```

---

# Основной формат

Использовать

```text
SVG
```

PNG использовать только:

- для предварительного просмотра;
- если SVG не поддерживается;
- для презентаций;
- для README.

# Запрещается

- менять форму логотипа;
- менять цвета;
- использовать символ пера;
- использовать сторонние версии логотипа;
- растягивать изображение;
- изменять пропорции;
- добавлять эффекты;
- изменять градиенты.

---

# Разрешается

- уменьшать;
- увеличивать;
- использовать SVG;
- экспортировать PNG необходимых размеров.

---

# Для Codex

Использовать только официальные файлы из данной директории.

Если необходим новый размер логотипа — масштабировать существующий.

Не создавать новые варианты логотипа.

Не изменять фирменный стиль проекта.
